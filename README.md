# ALLCON — Septic, Grading & MX Track Building

A professional business website for **ALLCON**, a North Georgia contractor specializing in septic installation, land grading & clearing, drain systems, and motocross track building. Based in Gainesville, GA and serving communities across the Southeast.

**Live Site:** [allconga.com](https://allconga.com/)

-----

## About the Project

This is a static, multi-page marketing website built with HTML and CSS. It is designed to generate leads and showcase ALLCON’s three core service lines to residential and commercial customers across North Georgia.

-----

## Services Covered

**Septic Installation**
New systems, drain fields, inspections, repairs, and grease traps — residential and commercial.

**Grading, Land Clearing & Drain Systems**
Driveways, building pads, land clearing, dirt work, underground stormwater retention, and drain systems.

**MX Track Building**
Backyard practice tracks, competition-level layouts, jump building, track reshaping, and drainage shaping. Travels across the Southeast.

## Getting Started

No build tools or dependencies required. Open any `.html` file directly in a browser, or serve locally with:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Deployment

The site includes a `CNAME` file, indicating it is deployed via **GitHub Pages** with a custom domain. It can also be hosted on any static hosting platform:

- [GitHub Pages](https://pages.github.com/) ← currently in use
- [Netlify](https://netlify.com)
- [Vercel](https://vercel.com)
- Any standard web host

To update the custom domain, edit the `CNAME` file in the repo root.
